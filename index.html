Privacy Policy – Face-T Extension

Last updated: April 23, 2026

1. Overview

Face-T Extension is an enterprise-grade browser extension that captures user interaction events and page screenshots to help teams analyze UX navigation within their own web applications. This policy describes what data is collected, how it is used, and where it is sent.

2. Data Collected

  • Web page URLs – used to verify whether the current page is on the organization's allow-list policy.
  • User interaction events – click and Enter key press events are recorded to mark the timing of screenshots.
  • Page screenshots (t0 / t1) – a "before" screenshot is captured at the moment of interaction, and an "after" screenshot is captured once the page finishes rendering.

3. How Data is Used

All collected data is used solely to analyze UX navigation patterns within the organization's web application. The data is never used for advertising, profiling, or any purpose unrelated to the extension's core function.

4. Data Transmission

All data (events and screenshots) is transmitted exclusively to a locally running Face-T agent on the user's own machine (default endpoint: http://127.0.0.1:17777). No data is sent to external servers, cloud services, or any third party.

5. Data Retention

The extension itself does not store any data persistently beyond extension settings (backend URL, agent URL). All screenshot and event data retention is managed entirely by the local Face-T agent and backend, under the control of the enterprise operator.

6. Third-Party Sharing

We do not sell, trade, or transfer any user data to third parties. Data does not leave the user's local machine.
